# Field
This is a horror game, which I made for "Pygame Community Halloween Jam 2022".

The game is very simple to play. Players need to use the direction keys to control the movement of the protagonist. Players can also get items on the map and put them into the package, and click to use them.

The theme of the jam is "Near Sighted/Blind", so players need to get firewood to light the torch when moving. Otherwise, the surroundings will become darker and darker!

Finally, you should escape from this cemetery!

The inspiration contributor of the game is WuxiaScrub. The game was designed by imitating his "Wuxia World" game with permission.
